# Changelog

## [v4.0] — 2026-06-16 — InkOS 融合版 🚀

### ✨ 新增：书籍管理
- **创建/列表/删除书籍项目**：每本书有题材、平台、主角、设定、写作规则
- **书籍详情页**：按编号排序的章节列表 + 一键推倒 + AI写作

### ✨ 新增：AI 写作代理（InkOS 灵感）
- **基于配置的 AI 提供商自动写章节**：2000-3000字/章
- **题材自适应模板**：玄幻/都市/言情/仙侠/科幻/灵异 6种写作指令
- **写作规则注入**：用户自定义写作铁律（开篇冲突/对话比例等）
- **章节大纲支持**：提供大纲让AI按剧情线写作
- **Token 消耗追踪**：每次AI写作自动估算并记录

### ✨ 新增：开书推倒系统
- **8大题材模板**：玄幻/仙侠/都市/科幻/言情/耽美/灵异/游戏
- **5维评分体系**：字数×15% + 去AI味×20% + 对话密度×15% + 钩子密度×25% + 开篇冲击力×25%
- **平台匹配推荐**：起点/飞卢/晋江/番茄 自动推荐最佳首发平台
- **钩子命中分析**：显示已命中/缺失的题材钩子
- **智能修改建议**：针对低分维度的具体改进方案

### ✨ 新增：InkOS 集成检测
- **启动时自动检测 InkOS CLI**：`inkos --version` 检查
- **Agent 状态看板**：5个Agent（Writer/Analyst/CoverArtist/InkosBridge/BookManager）
- **Terminal 状态栏**：左上红黄绿灯 + 右上AI/InkOS连接状态 + Token显示

### 🎨 UI 全面重设计（InkOS 终端美学）
- **Terminal 风格头部**：红黄绿灯 + 主机名 + 状态指示器
- **5Tab 导航重构**：工作台/编辑器/推倒/AI写/设置
- **Agent 状态卡片**：每个Agent的实时状态和指标
- **书籍卡片列表**：显示章节数/字数/题材/平台
- **推倒结果可视化**：环形评分图 + 分维度得分条 + 建议列表
- **AI写作进度**：实时进度条 + 章节预渲染 + 一键复制到编辑器
- **设置页重构**：提供商列表+测试+Token统计+Inkos状态+系统信息

### 🔧 技术改进
- **server.py v4.0.0**：952行（v3.5的492行→翻倍）
- **新增 15+ API 端点**：书籍CRUD/AI写作/开书推倒/Agent状态/写作记忆
- **index.html v4.0.0**：纯前端SPA，无外部依赖
- **完整错误处理**：所有API带错误提示和状态码
- **写作模板系统**：6种题材的AI写作指令模板

## [v3.0] — 2026-06-16 — 移动优先 + 服务端部署版 📱

### 移动端重构
- **底部导航栏**：5个Tab（写作/分析/模块/章节/设置），手机端核心交互
- **全屏编辑器**：移动端键盘自适应，safe-area适配刘海屏
- **弹出面板**：slide-up动画，分析/模块/章节/设置以面板形式呈现
- **触控优化**：44px+最小触控区域，`:active` 状态反馈
- **PWA支持**：manifest.json，可添加到手机主屏幕

### 后端部署升级
- **环境变量配置**：`HOST` `PORT` `DATA_DIR`
- **CORS中间件**：支持跨域访问
- **模块自动发现**：多路径扫描（本地/数据目录/home目录）
- **健康检查**：增强 `/api/health` + 新增 `/api/config`
- **章节删除API**：`DELETE /api/sessions/{filename}`

### 部署文件
- Dockerfile + docker-compose.yml
- systemd 服务文件 (workbench.service)
- Nginx 反向代理配置
- 一键部署脚本 (deploy.sh)

### 设计升级
- 移动端：底部Tab + 弹出面板
- 桌面端：保持侧栏 + 固定面板
- 响应式三档：手机 < 768px < 桌面
- 安全区域适配：`env(safe-area-inset-*)`
- 状态栏精简：chars/段落/对话比/AI等级
