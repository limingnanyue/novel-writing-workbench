# 鲁班打磨报告 · novel-writing-workbench

## 1. 验料结果

**真实问题：成立** — 小说作者需要一站式Web工具，不用离开浏览器就能完成写作+分析+查阅模块

**独特角度：40写作模块+实时AI检测+暗色奢华UI在Web端的融合** — 目前市面无同类产品

**安装理由：** `python3 server.py 8080` 即开即用，不需要安装复杂依赖

**公共传播性：** 强 — 截图即可展示，暗色蓝调 UI 自带传播属性

## 2. 访行记录 — InkOS 设计借鉴

| InkOS 特性 | 借鉴到工作台 | 改造方式 |
|:-----------|:------------|:---------|
| 多Agent流程(write→review→audit→revise) | 写作阶段指示器 | 顶部4阶段进度条 |
| 章节编号系统(0001_章名.md) | 自动章节编号 | 新建时自动+1 |
| 状态追踪(progress/state-degraded) | 写作目标追踪 | 实时目标进度圈 |
| 清晰的工作流阶段 | 写作流程面板 | 编辑器上方相位切换 |
| 模块化工具链 | 工具面板重构 | 更清晰的左侧工具分类 |
| 暗色终端美学 | UI 升级 | 融入 InkOS 终端感 |
